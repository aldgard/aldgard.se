JavaScript

JavaScript är det som jag vill beskriva som ledsagaren genom rummen. Den som ger besökaren en liten interaktiv rundvisning genom att t.ex. dölja eller ta fram innehåll beroende på var i rummet besökaren är. Kanske ger en mer smooth vistelse genom lite mjuka övergångar mellan sidor eller ser till så istället för kallt hårt och snabbt komma till en intern länk på sidan så blir det en mjuk slide till länken. Även bildspel skapas genom att koda i JavaScript samt ha koll på om besökaren faktiskt har fyllt i vissa fällt i ett formulär som skaparen vill att besökaren ska fylla i. Men JavaScript kan även användas för att skapa bildbehandlingssidor eller spel. Men skriptet kan även användas utanför webbläsaren i form av t.ex. node.js som tillåter utvecklaren att använda JavaScript på serversidan för att kunna arbeta med anslutningar till databaser, skicka e-post och liknande.

Själva JavaScript är ett så kallat DOM skript. DOM står för Document Object Model och kan kortfattas beskrivas som en plattforms- och språk-neutralt gränssnitt som tillåter program och script att dynamiskt få tillgång och uppdatera innehåll, struktur och stil av dokument.

När vi använder oss av JS (JavaScript) på sidan så måste vi tänka på vad syftet är. Vi kan göra besöket lättare, snyggare och smidigare med hjälp av JS men vi kan också göra sidan tung och rörig om vi använder förmycket JS. För att inte göra kodningen för tung och svår så har JS ett bibliotek med funktioner och kodrader som man kan hänvisa till i sin egen kod. På så sätt slipper du som utvecklare att skriva ner alla funktioner om och om igen. För att vi ska kunna använda oss utav det biblioteket måste vi visa webbläsaren vilket bibliotek vi vill använda. Detta gör vi genom att lägga en länk till biblioteket i vår head-tagg. Detta ser ut så här:

<head>

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>

</head>

När jag sedan i mitt html-dokument lägger till min JS-kod så gör jag det innanför html-taggen och med taggen <script> före och </script> efter. Så en JS-kod för att scrolla ner till interna länkar mjukt och smidigt ser ut så här:

<script>

$(document).on('click','a[href^="#"]', function(event){

event.preventDefault();

$('html, body').animate({

scrollTop:$($.attr(this, 'href')).offset().top}, 500);

});

</script>
